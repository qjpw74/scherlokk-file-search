## 为什么 Spotlight 不够用

说正事之前，先聊一下为什么需要 Scherlokk 这样的工具。

macOS 自带的聚焦搜索（Spotlight）用的是索引技术。意思是：它只认识它建过索引的文件。如果某个文件刚刚复制过来，或者放在系统不常扫描的目录里，索引还没更新，Spotlight 就找不到。而且 Spotlight 对隐藏文件、系统目录基本不管，PDF 里的具体内容也不是每次都能搜到。

Scherlokk 的做法不同。它直接扫磁盘上的真实文件，不依赖索引。你要找什么，它就实际去翻——本地磁盘、USB 移动硬盘、网络驱动器（AFP、SMB、FTP），都能搜。

这个区别听起来不大，但用起来体感差很多。

## 核心功能逐项拆解

### 文件快速搜索：两种模式对应不同场景

Scherlokk 提供了**简易模式**和**专业模式**两套搜索界面。

简易模式适合日常快速查找：输入文件名关键词或文档内容关键词，软件扫全盘可访问区域，结果很快出来。对于大多数"我知道文件大概叫什么"的情况，这个模式就够了。

专业模式就丰富多了。它额外支持：多目录并行搜索、排除自定义目录、主动搜索隐藏文件和文件标签、按文件大小/类型/日期筛选，还支持正则表达式匹配。如果你需要找"过去三个月改过的、文件名包含 report 的 .xlsx 文件"，专业模式一个查询就能搞定。

### 全文内容检索：PDF 里的字也能找

这个功能是很多用户选 Scherlokk 的核心理由。

不只是搜文件名，软件内置了主流格式的解码器，可以直接搜索文档正文内容，支持的格式包括 PDF、DOC、DOCX、RTF、RTFD、XLS、XLSX 等。其他格式也可以进行原始内容搜索。

对于整理学术资料或者手头有几百份合同 PDF 的人来说，这个功能直接解决了"知道文件里有这句话但找不到在哪个文件里"的问题。

### 多视图展示：看清文件在哪里

搜索结果支持三种视图切换：

- **列表视图**：快速浏览所有结果的文件名和详情
- **树状图视图**：看文件在目录结构里的具体位置
- **图标视图**：图片类文件可以直接预览缩略图

这三种视图的意义在于：搜出来一堆同名文件时，树状图能帮你一眼看出它们分别在哪个文件夹里。

### 文件/目录比较：不用再专门装 Beyond Compare

这是 Scherlokk 另一个让人惊喜的功能。

文件比较支持文本对比和二进制对比两种方式。文本对比会高亮显示两个文件之间逐行的差异；二进制对比可以判断两个文件是否真正相同（文件名相同不代表内容相同）。软件使用复杂算法在文件间查找相似块，以差异列表形式呈现结果。

文件夹比较功能则是把两个目录的结构和内容放在一起对比，差异部分会被标记出来。对于版本管理、备份核验、代码归档来说，这个功能非常实用。

### 高性能模式

当你预期搜索结果会非常多时，可以一键切换到高性能模式，软件会自动调整资源分配来加快搜索速度。即使同时开着多个应用，Scherlokk 的搜索速度也维持得不错。
